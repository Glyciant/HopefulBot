Format Description Information
$(user) Returns the user who triggered the bot event. See More
$(nick) Returns the user who triggered the bot event, but will use a custom nickname if possible. See More
$(level) Returns the user level of the user who triggered the bot event. See More
$(channel) Returns the channel the bot event occured in. See More
$(params(param)) Returns the command argument specified by param. See More
$(randInt(min,max)) Returns a random number between min and max. See More
$(result) Returns the result of a spam protection action - either Purge or Timeout. See More
